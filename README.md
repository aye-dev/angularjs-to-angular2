# AngularJS 1.5, Angular 8 and webpack 4 hybrid app seed.

## Based on:
```
https://github.com/angular/angular-seed
```
## Setup
```
npm i
npm i http-server -g
```
### Terminal 1
```
npm run build
```
### Terminal 2
```
npm run start
```
